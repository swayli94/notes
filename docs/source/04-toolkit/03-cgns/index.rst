CGNS
===========================

`CFD General Notation System (CGNS) <https://cgns.github.io/>`_ 
是 CFD 软件中非常重要的数据格式，得到越来越广泛的应用，越来越多的软件提供这方面的数据格式。

The CGNS provides a general, portable, and extensible standard for the 
storage and retrieval of computational fluid dynamics (CFD) analysis data.

    `A User's Guide to CGNS <https://cgns.github.io/CGNS_docs_current/user/index.html>`_
